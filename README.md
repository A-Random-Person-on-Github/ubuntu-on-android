## Installation

```bash
# download installer
curl -L -o install.sh https://bit.ly/udroid-installer
# run the installer
bash install.sh
# Install udroid xfce4 with this command
udroid -i xfce4
# Login to xfce4
udroid -l xfce4
```

for more see [here.](md/installation.md)
